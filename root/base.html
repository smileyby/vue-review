# 基础语法

  1. 虚拟dom，就是用普通的js对象去模拟 真实的dom

  2. 指令：凡是 v- 开头的行内属性 我们都称为vue指令

    1. v-model：主要应用于表单元素 value属性和input事件的一个合体
    2. v-text：解决小胡子语法的显示问题
    3. v-html：等同于原生innerHTML
    4. v-cloak：解决小胡子语法的显示问题
    5. v-pre：优化性指令（告诉vue 该指令下的所有元素不需要编译）
    6. v-if、v-else-if、v-else、v-show
      1. v-if 有比较大的初始加载开销（整体结构不加载）
      2. v-show 有比较大的切换开销 （display:none）
    7. v-for 循环 数组、对象、字符串和数字
    8. v-bind:变量名 => :变量名、:key=> 提升渲染性能
      1. 事件修饰符：
        1. .stop 阻止时间冒泡
        2. .prevent 组织默认事件
        3. .capture 在捕获阶段执行
        4. .self 只在点击当前元素的时候才会执行，冒泡传播不执行
        5. .once 只会执行一次
        6. .passive 优先执行默认行为还是绑定事件
      2. 按键修饰符
        1. .enter
        2. .tab
        3. .delete
        4. .esc
        5. .space
        6. .up
        7. .down
        8. .left
        9. .right
      3. v-model
        1. .lazy
        2. .number
        3. .trim
    9. :class='{active: true, normal: false}'
    10. v-on:事件名（v-on:click）

  3. definProperty 实现 数据双向绑定，vue3 采用Proxy

    function definep(obj, key, value) {
    Object.defineProperty(obj, key, {
        get: function proxyGetter(){
            return value

# JSX

export default {
    data(){
        return {
            name: 'aa'
        }
    },
    render(h){
        return h('h1', {
            class: 'box', 
            attrs: {id: 'app'}
        }, [
            h('input', {
               domProps: {
                   value: this.name
               },
               on: {
                   input: function(e){
                      this.name = e.target.value;
                   }
               } 
            }),
            this.name,
            h('i', this.name)
        ])
    }
}

class VueRouter {
    constructor(options){
        let { routes } = options;
        this.routerMap = {}
        routes.forEach((item) => {
            this.routerMap[item.path] = item.component;
        })
        Vue.util.defineReactive(this, "_route", "/");
        this._route = location.hash.slice(1);
        window.onhashchange = ()=>{
            this._route = location.hash.slice(1);
        }
    }
}
VueRouter.install = install;
export default VueRouter;
